Beban Pencemar
Hitung BPM (Beban Pencemar Maksimum) dan BPA (Beban Pencemar Aktual) berdasarkan debit sungai, baku mutu, dan konsentrasi aktual per parameter.
BPM = Q × BM × 3.6 kg/jam

Usulan Baku Mutu
Hitung konsentrasi maksimum yang diperbolehkan dari sumber air limbah berdasarkan daya tampung sungai dan debit efluent.
Cmaks = BM × (Qs + Qe) / Qe

Alur Kerja
01
Input Data Sungai
Masukkan nama sungai, kelas mutu, dan debit aliran (m³/s) untuk musim kemarau dan hujan. Atau import otomatis dari file Excel hasil uji lab via fitur AI.
02
Pilih Parameter & BM
Pilih parameter dari database 49 parameter. Nilai baku mutu terisi otomatis sesuai kelas sungai PP 22/2021. Masukkan konsentrasi aktual hasil uji laboratorium.
03
Hitung & Analisis
BPM dan BPA dihitung otomatis. Status tiap parameter (memenuhi/melebihi baku mutu) ditampilkan. Jalankan pemodelan dispersi untuk proyeksi sebaran pencemar.
04
Export Laporan
Generate laporan Excel berwarna dan Word terstruktur. Simpan proyek ke riwayat browser atau ekspor ke file .riverload untuk dibagikan ke rekan kerja.

Usulan Baku Mutu Maksimum Air Limbah
FORMULA
Cmaks = [ BM × (Qs + ΣQi) − Qs × Chulu ] / ΣQi
Nilai ini adalah konsentrasi maksimum air limbah yang masih dapat dibuang sehingga CR hilir sungai tepat sama dengan Baku Mutu kelas sungai yang dipilih.
Diambil nilai terkecil antara musim kemarau dan hujan (kondisi kritis). Jika Cmaks < 0 → konsentrasi hulu sudah melampaui BM, tidak ada ruang pencampuran.
